 1 уровень сложности: Написать псевдокод, блок-схему или текстовое описание для алгоритма: сложить три числа и вывести сумму.
Реализовать алгоритм в коде.

Описание алгоритма для сложения трех чисел и вывода суммы:

1.Введите первое число и сохраните его.
2.Введите второе число и сохраните его.
3.Введите третье число и сохраните его.
4.Вычислите сумму трех чисел, сложив их.
5.Выведите полученную сумму на экран.

 Реализуем этот алгоритм на примере JavaScript:

// Шаг 1: Ввод трех чисел
const number1 = parseFloat(prompt("Введите первое число:"));
const number2 = parseFloat(prompt("Введите второе число:"));
const number3 = parseFloat(prompt("Введите третье число:"));

// Шаг 2: Вычисление суммы
const sum = number1 + number2 + number3;

// Шаг 3: Вывод результата
console.log("Сумма трех чисел:", sum);
