Описание задания.
  Реализуйте и оцените алгоритм поиска максимального числа в неотсортированном массиве (объясните почему именно такая оценка). Реализовать можно как в виде текста, псевдокода или блок-схемы, так и в виде реального кода

Описание алгоритма поиска максимального числа в неотсортированном массиве:

1.Задать переменную maxNumber для хранения максимального числа и инициализировать её значением первого элемента массива.
2.Пройтись по всем остальным элементам массива.
3.Для каждого элемента:
Если текущий элемент больше значения переменной maxNumber, обновить значение maxNumber этим элементом.
4.По окончании прохода, значение переменной maxNumber будет содержать максимальное число в массиве.

Оценка алгоритма:

Сложность данного алгоритма поиска максимального числа в неотсортированном массиве - O(n), где n - количество элементов в массиве. Это объясняется тем, что мы выполняем один проход по всем элементам массива и сравниваем каждый элемент с текущим максимальным числом. В худшем случае (когда максимальное число находится в самом конце массива или в случае равных чисел) нам придется выполнить n - 1 сравнение.

function findMaxNumber(arr) {
  if (arr.length === 0) {
    return undefined; // Возвращаем undefined для пустого массива
  }
  
  let maxNumber = arr[0];
  
  for (let i = 1; i < arr.length; i++) {
    if (arr[i] > maxNumber) {
      maxNumber = arr[i];
    }
  }
  
  return maxNumber;
}

const numbers = [12, 5, 27, 9, 3, 18];
const maxNumber = findMaxNumber(numbers);

console.log("Max number:", maxNumber); // Max number: 27
